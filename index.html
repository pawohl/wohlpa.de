AMA-App

Drugs and other chemicals can be identified by numeric or boolean properties. AMA-App is a platform that was written for allowing drug identification through Thin-Layer-Chromatography (TLC) although it is generic enough to support any other analytical methods. For the end user, it comes with a deadly-simple-to-use and to host web application allowing the search for compounds by experimentally obtained properties.

Skin cancer database

Some environment and working condition factors are known to cause an increase in skin cancer at specific skin regions. When physicians diagnose skin cancer, they have to specify the body region. Unfortunately this classification is rather fuzzy and so are studies that use these data as their source. That's why we at Martin-Luther-University built an application that allows selection in an interactive 3D model. Besides many other advantages, the results will be more reproducible and when the patient revisits the physician or a colleague with the same model they can spot the lesion more safely while still being cost-effective.

opusenc.js

Most probably the code you need is already written, though maybe in a different programming language. The opusenc tool was written in C and I ported that to ECMA-Script (JavaScript) so it runs in every browser and you can now compress audio client-side before consuming your client's precious bandwidth. Cross-compiling with Emscripten and passion about translation and programming languages helped me a lot.

MolHandler

Structured data can be more easily processed and used by communities than bare images. Complex structures should be easily forkable and editable. With the help of some really cool mentors during a Google Summer of Code, I created MolHandler for MediaWiki, the software powering Wikipedia and many other Wikis. MolHandler allows rendering of Chemical Table Files to graphics. In future it could extract more information and provide that to MediaWiki.
